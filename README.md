#移动端地区选择控件
>这是一个简单的地区选择控件，用来在移动的有地区选择的业务时使用。

##how to start
>
>   var selectArea = new MobileSelectArea();
>   selectArea.init({trigger:'.region',
>                    value:$('#hd_area').val(),
>                    data:'data/datac.js'
>                });
                

```
   var selectArea = new MobileSelectArea();
   selectArea.init({trigger:'.region',
                    value:$('#hd_area').val(),
                    data:'data/datac.js'
                });
```
